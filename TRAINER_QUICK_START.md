# Trainer Quick Start (One-Page)

## Goal
Run a complete batch session in 10-15 minutes setup time.

## 1. Login As Trainer
1. Open app.
2. Enter `Name`.
3. Select role `Trainer`.
4. Enter `Trainer Key`.
5. Click `Start`.

## 2. (Optional but Recommended) Verify Admin
1. In `Admin Access Control`, enter `Admin Key`.
2. Click `Verify Admin`.
3. Click `Load Admin Data`.

Use this section to:
- Set trial question limit
- Update trainee access code
- Update trainer key

## 3. Create Cohort/Class
1. In `Cohort / Class Management` fill:
   - Cohort Name
   - Cohort Access Code
   - Cohort Question Limit
   - Active = Yes
2. Click `Create Cohort`.
3. Select cohort in `Existing Cohorts`.

## 4. Enroll Learners
1. Select cohort.
2. Enter learner email (required), name/phone (optional).
3. Click `Enroll / Update Member`.

## 5. Load Question Bank
1. Open `Trainer Deck Manager`.
2. Upload CSV/XLSX OR paste data.
3. Click `Import CSV File` or `Import CSV`.

Preferred headers:
`tag,type,question,answer,option_a,option_b,option_c,option_d,correct_option`

## 6. Assign Exam Blueprint To Cohort
1. In `Exam Blueprint Assignment`:
   - Select blueprint template
   - Adjust questions/duration/pass threshold if needed
2. Click `Assign to Selected Cohort`.

Default templates:
- ICD-Heavy Mock
- CPT-Heavy Mock
- Mixed Final Mock

## 7. Tell Learners How To Join
- Choose role `Trainee`
- Enter cohort access code
- Start session

If blueprint is assigned, trainee exam settings auto-apply and lock.

## 8. Monitor Progress
Use:
- `Trainer Session Console` -> session-level performance
- `Weak-Topic Analytics` -> user/batch accuracy trends + weak tags

Buttons:
- `Load User Analytics`
- `Load Batch Analytics`
- `Recommend Daily Drill`
- `Share Trend by Email`

## 9. Export Completion Proof
Use top button `Export PDF Report` after a session to generate:
- Score summary
- Attempted/correct/wrong
- Date/time
- Category breakdown
- Pass/fail vs threshold

## 10. End-Of-Day Checklist
- Refresh sessions and analytics
- Share trend report emails
- Export key learner reports (PDF)
- Keep cohort access codes private
- End trainer session
